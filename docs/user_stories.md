# Пользовательские истории и роли

## Роли

| Роль       | Описание                                                                 |
|------------|--------------------------------------------------------------------------|
| `guest`    | Неавторизованный посетитель сайта                                        |
| `employee` | Зарегистрированный сотрудник компании                                    |
| `admin`    | Администратор: управляет справочниками месторождений и скважин           |

## Матрица прав

| Действие                                        | guest | employee | admin |
|-------------------------------------------------|:-----:|:--------:|:-----:|
| Просмотр главной страницы                       |   +   |    +     |   +   |
| Просмотр каталога месторождений и карточек      |   +   |    +     |   +   |
| Регистрация / Вход                              |   +   |    +     |   +   |
| Просмотр списка скважин                         |   −   |    +     |   +   |
| Создание заявки на обслуживание                 |   −   |    +     |   +   |
| Изменение статуса своей заявки                  |   −   |    +     |   +   |
| Изменение статуса любой заявки                  |   −   |    −     |   +   |
| CRUD месторождений                              |   −   |    −     |   +   |
| CRUD скважин                                    |   −   |    −     |   +   |

## User stories

### Гость (guest)
- **US-G-1.** *Как гость, я хочу видеть главную страницу с описанием компании,
  чтобы понять, чем она занимается.*
- **US-G-2.** *Как гость, я хочу просматривать список месторождений и их
  характеристики, чтобы оценить масштаб деятельности компании.*
- **US-G-3.** *Как гость, я хочу зарегистрироваться, чтобы получить доступ к
  служебным разделам сайта.*

### Сотрудник (employee)
- **US-E-1.** *Как сотрудник, я хочу авторизоваться по email и паролю, чтобы
  попасть в служебные разделы.*
- **US-E-2.** *Как сотрудник, я хочу видеть список всех скважин и фильтровать
  их по статусу, чтобы быстро находить нужные.*
- **US-E-3.** *Как сотрудник, я хочу создать заявку на обслуживание скважины с
  указанием приоритета, чтобы сообщить о выявленной проблеме.*
- **US-E-4.** *Как сотрудник, я хочу менять статус собственной заявки
  («в работе», «выполнено»), чтобы фиксировать прогресс.*

### Администратор (admin)
- **US-A-1.** *Как админ, я хочу добавлять, редактировать и удалять
  месторождения, чтобы поддерживать справочник в актуальном состоянии.*
- **US-A-2.** *Как админ, я хочу добавлять скважины к месторождениям и менять
  их параметры (глубина, добыча, статус), чтобы отражать реальную обстановку
  на промысле.*
- **US-A-3.** *Как админ, я хочу видеть все заявки и менять любой их статус,
  чтобы координировать работу бригад.*
- **US-A-4.** *Как админ, я хочу удалять заявки, потерявшие актуальность,
  чтобы не захламлять интерфейс.*
